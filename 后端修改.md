### 2.3 接口详细设计

#### 2.3.1 文件上传

```
POST /api/common/upload
Content-Type: multipart/form-data
```

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `file` | File | 是 | 上传文件 |
| `module` | String | 否 | 模块名，如 `resume`、`employee`，用于分类存储 |

**响应：**

```json
{
  "code": 200,
  "message": "上传成功",
  "data": {
    "fileName": "abc123.pdf",
    "fileKey": "resume/2026/07/abc123.pdf",
    "fileUrl": "https://minio.xxx.com/hr-files/resume/2026/07/abc123.pdf",
    "fileSize": 102400
  }
}
```

**实现要点：**
- 上传至 MinIO，Bucket 名 `hr-files`
- 文件路径规则：`{module}/{yyyy}/{MM}/{uuid}.{ext}`
- 文件大小限制：10MB（与 `application.yml` 中 `spring.servlet.multipart.max-file-size: 50MB` 一致即可）
- 支持格式：`.pdf` `.doc` `.docx` `.xls` `.xlsx` `.png` `.jpg` `.jpeg`

#### 2.3.2 文件下载

```
GET /api/common/download/{fileKey}
```

**路径参数：**

| 参数 | 类型 | 说明 |
|------|------|------|
| `fileKey` | String | 上传时返回的 `fileKey`，如 `resume/2026/07/abc123.pdf` |

**响应：** 文件二进制流，`Content-Disposition: attachment`

**实现要点：**
- 从 MinIO 读取文件并写入 `HttpServletResponse.getOutputStream()`
- 设置正确的 `Content-Type`（根据文件扩展名）

#### 2.3.3 下拉选项

```
GET /api/common/options?type=education
```

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `type` | String | 是 | 选项类型 |

**type 枚举值：**

| type | 说明 | 返回值示例 |
|------|------|-----------|
| `education` | 学历 | `[{value:"bachelor", label:"本科"}, ...]` |
| `gender` | 性别 | `[{value:1, label:"男"}, {value:2, label:"女"}]` |
| `resume_source` | 简历来源 | `[{value:"boss", label:"BOSS直聘"}, ...]` |
| `leave_type` | 假期类型 | `[{value:"annual", label:"年假"}, ...]` |
| `transfer_type` | 异动类型 | `[{value:"regular", label:"转正"}, ...]` |

**响应：**

```json
{
  "code": 200,
  "message": "成功",
  "data": [
    { "value": "bachelor", "label": "本科" },
    { "value": "master", "label": "硕士" },
    { "value": "doctor", "label": "博士" }
  ]
}
```

**实现要点：**
- 可使用枚举类或字典表获取选项
- 建议创建 `SysDict` 字典表统一管理，支持动态配置

## 六、开发建议

### 6.1 /common 模块实施步骤

1. **MinIO 集成**：确认 Docker Compose 中 MinIO 已正常启动，编写 `MinioTemplate` 工具类
2. **文件上传**：创建 `CommonController`，实现 `POST /common/upload`
3. **文件下载**：实现 `GET /common/download/{fileKey}`
4. **下拉选项**：建议创建 `sys_dict`（字典表）和 `SysDictController` 或在 `CommonController` 中实现
5. **权限配置**：在数据库中为 `/common` 接口配置相应权限标识

### 6.2 文件存储结构建议

```
MinIO Bucket: hr-files
├── resume/          # 简历附件
│   └── 2026/07/
├── employee/        # 员工导入导出
│   └── 2026/07/
├── document/        # 办公文档
│   └── 2026/07/
├── notice/          # 公告附件
│   └── 2026/07/
└── avatar/          # 用户头像
    └── 2026/07/
```

### 6.3 依赖确认

- MinIO Java SDK 已加入 `pom.xml`（`io.minio:minio`）
- `application.yml` 中已配置 `spring.servlet.multipart`（max-file-size: 50MB, max-request-size: 100MB）
- 需补充 MinIO 连接配置（endpoint、accessKey、secretKey）
